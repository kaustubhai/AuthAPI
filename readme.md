# Node Authentication API
### A full fledged, API for handling users and there authentication  


API Offers endpoints for
- [x] Register User
- [x] Login User with PIN
- [x] Login User with password
- [x] Forgot Password
- [x] Load User Details
- [x] Update user Name
- [x] Update user Mail
- [x] Reset User's Pin
- [x] Reset User's Password
